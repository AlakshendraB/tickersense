# TickerSense

TickerSense is a web-based stock analytics platform that provides real-time market visualizations and AI-generated insights.

## ✨ Features

- 📊 Interactive Chart.js graphs (Line, Bar, Pie)
- 🧠 AI-powered summaries for historical data
- 🔍 Search with autosuggestions
- 📌 Watchlist saved to localStorage
- ☁️ CSV upload via Papa Parse
- 🌙 Responsive dark-themed Bootstrap UI
- ⚡ Real-time loading states and error handling

- JavaScript (ES6)
- Bootstrap (Dark theme)
- Chart.js
- PapaParse
- LocalStorage API

1. Clone this repo
2. Open `index.html` in a browser
